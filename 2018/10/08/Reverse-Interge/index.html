Reverse Interge

2018-10-08
LeetCodeEasy
mod

Description

Given a 32-bit signed integer, reverse digits of an integer.

Example 1

Example 2

Example 3

Solution

MOD

The idea is to get the least digit and cut it off from the given number by “mod” in a loop.

Example:

$$123 \to 12 \to 1$$
$$3 \to 3\times10+2 \to 32\times10+1$$

String

There is another tricky way to use the convenice of string utils in python by simply reversing a string and then convert it to int.
